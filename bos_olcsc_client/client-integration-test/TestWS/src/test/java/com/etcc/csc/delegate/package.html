This package tests the Delegate objects. By default, it will test the Delegates in their "WebServices" mode. To test a specific mode (such as EJB, WebServices, etc.), see the documentation for com.etcc.csc.common.ServiceFactory and com.etcc.csc.common.ServiceFactoryTest. Note that there is also a "Demo" mode, using;

ETCC_HCTRA_SERVICE_FACTORY_IMPL = com.etcc.csc.common.ServiceFactoryTestImpl

either as an environment variable, or a command line property. In "Demo" mode, there is no Database access, but instead hardcoded values from the test libraries are used. Additional test libraries can be added to the war to demonstrate additional pages.
